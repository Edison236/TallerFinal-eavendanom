Documentación de la API REST

Esta es la documentación de la API REST para gestionar una guarderia.

Introducción

Esta API permite realizar operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre los usuarios.

Nota:

En el tema Web se uso para e despliegue de PYTHON: https://vercel.com/
Para el tema de base de datos se uso: https://railway.app/

Endpoints:

  • Local:http://127.0.0.1:5000/
  • WEB:https://apirest-eavendanom.vercel.app

2. Actualizar el nombre de animal

URL: /daycare/update_animals/id

Método: PATCH

Ejemplo de body


        {
            "name": "Nuevo nombre"
        }

Ejemplo de respuesta:


        {
            "mensaje": "Datos actualizados correctamente."
        }

3. Borra un animal por id

URL: /daycare/delete_animals/id

Método: PATCH

Ejemplo de respuesta:


        {
            "mensaje": "El animal '{nombre de animal}' se borro correctamente."
        }

4. Crea un nuevo animal

URL: /daycare/create_animal

Método: POST

Ejemplo de body


        {
            "name" : "Copito",
            "type" : "Gato",
            "breed": "Persa",
            "age"  : 3,
            "sound": "Miaow"
        }

Ejemplo de respuesta:


        {
            "mensaje": "El animal Copito fue creado correctamente."
        }
